If you'd like to add more content to the vault, it should be fairly simple. Just find the .md files for the content you're bringing in, drag them to a folder, and boom; they're in the vault. See [[Tips and Tricks]] for a list of a number of files that just have to have the same name and, thus, their backlinks might break if you reorganize. Keep in mind if you add anything to the vault with the same name as an existing file, it will also potentially mess up backlinks. Here are some folder details of what type of content you put in those folders and the templates they populate with to make transferring content in at the speed of thought.
#### Backgrounds
The notes in this folder automatically populate with the [[Background]] template to make copying information over very easy if you need to copy it in from a book or DnD Beyond.
#### Classes
The notes do not automatically populate with the [[Class]] template since most of the files in here are going to be class features, but you can copy/paste from that template or use "Alt+T" to quickly add the class template. Realistically subclasses are the only thing you should have to add for the core 12 classes, but you might also want to pull in things like the Artificer, Blood Hunter, or the wonderfully crafted Mystic Arts Warlord. For that last one, you should be able to just create a Warlord folder, drag the .md file in from the Mystic Arts Patreon, and it'll populate as a folder note. From there you can create subclass folders, subclass documents, class feature documents, etc.
#### Equipment
The default template here is [[Equipment]], but there are a few special Equipment types such as [[Magic Item]] and [[Tool]] as well. If you wish to use the Mystic Arts Arcane Forge, I highly recommend renaming the official Adamantine and Mithral equipment to Flawed Adamantine and Flawed Mithral since Arcane Forge includes its own higher rarity versions of this very equipment. Could also just call the Arcane Forge stuff Mystic Adamantine and Mystic Mithral if you wanted.
#### Feats
Automatically populates with [[Feat]] for easy copying and is split into folders. In my own personal vault, I have the Dragonmarked Feats from *Forge of the Artificer* as their own folder, but you could decide to just plop them into Origin Feats if you want.
#### Lore
This is just a blank folder to fill with all those lore goodies.
#### Rules
If you've got homebrew conditions, actions, etc. plop em here. The Homebrew Systems folder is also where you can toss those funky Mystic Arts .md files for Hexploration, Arcane Forge, etc.
#### Species
Automatically populates with [[Species]] template for easy copying.
#### Spells
Automatically populates with [[Spell]] template for easy copying.
#### Stat Blocks
Automatically populates with [[Creature]] template for easy copying. I've also done the honors of adding a blank Homebrew folder for ya so you can keep your own creations separate from official. Mystic Arts gets its own special separate folder where you can put all those wonderful Dadi goodies such as the Lizardfolk, Goblins, and Sahuagin.
#### z. Load Bearing Folders
Templates and Attachments go here in their respective folders.
