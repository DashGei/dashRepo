#### Anchor Token [[Anchor Token|(200 GP)]]
*Wondrous Item, Uncommon*

**Attunement:** No

This object looks like a feather. You can take a [[Magic]] action to touch the token to a boat or ship. For the next 24 hours, the vessel can’t be moved by any means. Touching the token to the vessel again ends the effect. When the effect ends, the token disappears.
